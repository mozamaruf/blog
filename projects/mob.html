ABOUT THE PROJECT

The project is focused on creating awareness theme which focuses on designing a mobile web experience for users to experience enhanced awareness on sustainable practices. In order to solve the big issue of product obselence, I propose a solution of designing and prototyping a mobile application which will help users to share a used product with other users nearby.
So for example, an average use of drill machine in it’s entire life is used somewhere between 10 to twenty minutes. This doesn’t require every user to buy a drill machine, instead they could share with each other and that’s the main goal of the application. The solution exists because there doesn’t exist many applications based on this sharing economy.

Tools used for the application includes: Jquery, HTML & Javascript

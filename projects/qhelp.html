ABOUT THE PROJECT

QatarHelp is a platform which let's customer errands to be performed by "skilled-taskers". Customer post a request for any task to be completed. Service providers accept or negotiate on the price before completing the task. The project went on to win Runner's-up for CMU-Q Project Plan Competition.

Tools required for the platform includes: NodeJs, Sql, HTML, Bootstrap, Git, Heroku
